# 基本语法

```shell
sed [options] {sed-commands} [input-file]...  
```

sed处理文本以**行**为单位。sed从input-file里读取第一行，然后对该行执行sed-commands，然后输出到标准输出（从输入到输出，只处理，不改变input-file）。重复，直到所有行读完。
