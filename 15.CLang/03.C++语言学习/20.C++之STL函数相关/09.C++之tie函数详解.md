# 概述

tuple 即元组，可以理解为pair的扩展，可以用来将不同类型的元素存放在一起，常用于函数的多返回值。

tuple可以使用初始化列表进行赋值。

```c++
tuple<int,double,string> t3 = {1, 2.0, "3"};
```
