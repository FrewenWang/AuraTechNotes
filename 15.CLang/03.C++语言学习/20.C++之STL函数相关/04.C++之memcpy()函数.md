# 概述

文章参考：https://www.runoob.com/cprogramming/c-function-memcpy.html

## 概述

```c++
#include<string.h>

void *memcpy(void*dest, const void *src, size_t n);
```

功能

由src指向地址为起始地址的连续n个字节的数据复制到以destination指向地址为起始地址的空间内。

函数返回一个指向dest的指针。
