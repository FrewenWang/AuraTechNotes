# 安装Cmake

## ubuntu环境

安装命令

```
sudo apt install cmake
```

**卸载原来的cmake版本:**

```
sudo apt-get remove cmake
```



## MacOS环境

下载地址：https://github.com/Kitware/CMake/releases/download/v3.26.5/cmake-3.26.5-macos-universal.dmg
