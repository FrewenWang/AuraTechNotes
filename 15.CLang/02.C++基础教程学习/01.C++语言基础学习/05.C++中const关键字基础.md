## 概述

​		定义变量时的限定符，表示变量值不能改变。

```c++
const int bufSize = 512;
bufSize = 512; // 错误：试图向const对象写值
```

​		由于const一旦创建就不可更改，所以**const对象必须初始化**（否则定义一个默认值且不可修改的变量没有任何意义）。

```c++
const int i = get_size(); // 正确
const int j = 42; // 正确
const int k;      // 错误：未初始化
```

​		使用**值传递初始化**时，被初始化的对象是否为const与初始化对象是否为const无关。也即，const对象与非const对象可以互为初始化。
