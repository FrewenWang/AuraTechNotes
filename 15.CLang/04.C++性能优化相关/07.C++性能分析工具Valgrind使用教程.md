# 概述



## CLion使用Valgrind

MacOS:

```
brew tap LouisBrunner/valgrind
brew install --HEAD LouisBrunner/valgrind/valgrind
```


在 mac 下安装 Valgrind 比较麻烦，在尝试多次后，总结出上面的安装方式

Linux:

```
sudo apt install valgrind
```


Windows，WSL 环境下:

```
sudo apt-get install valgrind
```
