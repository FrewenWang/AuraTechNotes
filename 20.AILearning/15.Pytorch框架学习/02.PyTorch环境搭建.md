# 测试环境

输出结果如下，表明环境安装成功：

```
from __future__ import print_function
import torch


def print_hi(name):
    # 在下面的代码行中使用断点来调试脚本。
    print(f'Hi, {name}')  # 按 Ctrl+F8 切换断点。


# 按间距中的绿色按钮以运行脚本。
if __name__ == '__main__':
    print_hi('Hello Pytorch World')
    print_hi(torch.__version__)
    x = torch.rand(5, 3)

    print(x)
```
