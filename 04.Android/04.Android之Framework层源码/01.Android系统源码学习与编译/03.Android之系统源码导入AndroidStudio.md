# 代码编译

```
source build/envsetup.sh


 lunch 5
 
 // 在编译之前，可以输入以下命令，查看你的Mac CPU核数：
 sysctl -n machdep.cpu.core_count
 
 // 输出为4核，我们就可以启动4个线程来编译源码：
 make -j4
```
